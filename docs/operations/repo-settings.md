# Repository Settings Checklist

Last reviewed: 2026-03-23

Keep these GitHub settings enabled for `landigf/JMCP`:

- branch protection on `main`
- pull request reviews for destructive or security-sensitive changes
- dependency graph and dependency alerts
- secret scanning and push protection
- GitHub Security Advisories
- Actions permissions defaulting to read-only when possible
- Code scanning enabled through the checked-in workflow
