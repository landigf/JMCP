# Privacy Baseline

Last reviewed: 2026-03-23

JMCP starts from a best-practice privacy baseline rather than a formal certification target.

## Baseline controls

- collect the minimum data needed to steer tasks
- avoid copying repository content into third-party systems unless necessary
- do not log secrets or authentication factors
- define retention and deletion expectations before adding telemetry
- document where user-visible summaries, screenshots, and traces are stored

## Current position

The repository is public, but the future runtime is intended for a single trusted operator. If JMCP later expands to a team or broader audience, the privacy model and legal basis will need to be upgraded in the same repository before the product scope expands.
